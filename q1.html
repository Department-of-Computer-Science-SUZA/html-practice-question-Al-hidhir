Java Program

        import java.util.Scanner;

public class Name {
    public static void main(String[] args) {
        
        Scanner s = new Scanner(System.in);
        
        System.out.print("Enter your name: ");
        
        String name = s.nextLine();

        System.out.println("Your name is: " + name);
    }
}

Explanation

This is a program that print your name. It consists of a single class named Jina. Inside this class, there is a main method which serves as the entry point for the program. The main method simply prints the string of your name that you enterd to the console using the System.out.println() statement.
